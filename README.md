# Calculadora Interativa | Interactive Calculator

Uma calculadora simples via linha de comando que realiza operações aritméticas básicas até que o usuário decida sair.
A simple command-line calculator that performs basic arithmetic operations until the user decides to exit.

---

## Português

### 📋 Descrição
Este programa solicita ao usuário que selecione uma operação aritmética (adição, subtração, multiplicação ou divisão) e insira dois números.
Em seguida, executa a operação escolhida e exibe o resultado. Esse processo roda em um laço de repetição contínuo até que o usuário selecione a opção "Sair".

### 🚀 Como Executar
1. Certifique-se de ter o **Python 3** instalado.
2. Execute o script no seu terminal:
   ```bash
   python calculadora.py
   ```
3. Escolha uma operação no menu (1 a 5).
4. Digite os dois números para obter o resultado.

## English

### 📋 Description
This program prompts the user to select an arithmetic operation (addition, subtraction, multiplication, or division) and input two numbers. 
It then executes the chosen operation and displays the result. This process runs in a loop and repeats continuously until the user selects the "Exit" option.

### 🚀 How to Run
1. Make sure you have **Python 3** installed.
2. Run the script in your terminal:
   ```bash
   python calculator.py
   ```
3. Choose an operation from the menu (1 to 5).
4. Enter the two numbers to get the result.
